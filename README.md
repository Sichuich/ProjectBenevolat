## Exécution avec Maven
<p>
  Pour exécuter l'application avec Maven, suivez les étapes ci-dessous :
    <ul>
        <li>mvn compile</li>
        <li>mvn exec:java</li>
    </ul> 
  Cela va compiler et exécuter l'App. 
</p>
